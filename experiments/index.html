Org chart visualization experiments

Four standalone takes on the org chart in org-chart-editable_3.html. Each shares the same employee schema and can swap in your real data via the "Drop CSV" input — export from the main app's Headcount Plan tab.

3D Force Graph

Nodes orbit in 3D space, manager→report links as edges. Best for spotting cross-team clusters and isolated branches. Drag to rotate.

three.js · 3d-force-graph

Radial Sunburst

CEO at center, each ring outward = one reporting level. Wedge size = subtree headcount. Best for showing org shape at a glance.

d3 partition

Whiteboard Sketch

Standard tree, but rendered hand-drawn — sketchy lines, post-it notes, marker fills. Reads as "draft / not final" instead of corporate.

rough.js · d3 tree

Subway Map

Vignelli-style transit diagram. Each department is a coloured line, each person is a station. Manager/report ties become transfer links.

d3 · custom routing
All four files load _shared.js for sample data and a CSV importer. Pick a winner and we can port it back into the main app as a new view.
